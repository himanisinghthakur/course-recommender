## AI Course Recommendation System

A command-line interface (CLI) project that recommends personalized courses based on the user's interest and learning stage using AI/ML concepts.
Built in Python, this project uses the rich library to provide a colorful, interactive, and visually appealing terminal experience.


## Features
- User Input: Select your field of interest and learning stage.
- KNN-Based Recommendation: Recommends the top 3 courses closest to user preferences.
- Weighted Skill Boost: Displays how suitable the recommended course is for the user.
- Top Courses Table: Highlights the best course visually.
- Career Guidance: Suggests real-life career paths based on courses.
- Interactive CLI:
      - Spinner animation simulating AI processing
      - Colorful text, panels, emojis, and bold formatting
      - Skill Boost progress bar for better visualization


## Installation
1. Clone the repository:
    git clone https://github.com/{username}/{repo-name}.git
    cd {repo-name}

2. Install dependencies:
    pip install rich
Requires Python 3.x




## How to Use
- Open terminal in VS Code (or any terminal).
- Run the Python script:
    python main.py
- Follow the prompts:
    - Select your interest (Coding, Design, Business, AI/Data Science, Marketing)
    - Select your learning stage (Beginner, Intermediate, Advanced)
- View results:
    - Top 3 recommended courses
    - Best course highlighted
    - Skill Boost percentage
    - Learning stage
    - Career suggestion
    - Explanation of why the course fits you

## Dataset

Courses cover Coding, Design, Business, AI/Data Science, and Marketing:
  - Beginner, Intermediate, and Advanced levels
  - Includes career guidance for realistic recommendations



## ML Concept
- K-Nearest Neighbors (KNN): Finds courses closest to user preferences
- Weighted Scoring: Assigns scores based on similarity to user interest and learning stage
